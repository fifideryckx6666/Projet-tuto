Installation d’un SCANNER

  • Aller ouvrir le fichier « MOPIEUR.docx Il s’agit du fichier envoyé dans OUTLOOK par Franck
  • Choisir lien de connexion direct : le coller dans la barre du navigateur choisi. Ex. : 5256V : https://172.22.92.21:wcd/top.xml (dgifip)
  • Menu de démarrage Cliquer sur « Administrateur » + « connexion » Mettre le mot de passe envoyé par le même message : Solimp3-n9IAAt
  • Une nouvelle page s’ouvre Cliquer sur « Enr. Dest. »
  • Une nouvelle page s’ouvre Renseigner le nom Renseigner l’adresse mèl
  • Un pop-up s’ouvre Cliquer sur « Poursuivre Enregistrement »
  • Pour résumer L’installation d’un scanner se fera par l’interface fournie par KONICA MINOLTA
